Sensitive Data Exposure

Concept

Applications may expose sensitive information unintentionally through misconfigured pages, verbose error messages, or publicly accessible configuration endpoints.

Examples

Common types of exposed data include:

  • Configuration files
  • API keys
  • Passwords

Impact

Attackers can use exposed credentials, API keys, or internal configuration details to gain unauthorized system access and escalate their attack.
