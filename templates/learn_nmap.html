Network Scanning with Nmap

Concept

Nmap (Network Mapper) is an open-source tool used for network discovery and security auditing. It is one of the most widely used tools in penetration testing.

Nmap can discover:

  • Open ports on a target machine
  • Running services and their versions
  • Operating system details
  • Hidden or unexpected services

Example Commands

Basic port scan:

nmap {{ target_host }}

Scan a specific port range:

nmap -p 5000-5100 {{ target_host }}

Service version detection:

nmap -sV {{ target_host }}

What To Look For

When scanning a target, look for:

  • Ports in OPEN state
  • Unexpected services running on non-standard ports
  • Internal services that shouldn't be publicly accessible

Real World Impact

Attackers use Nmap to map out a target's attack surface. Discovering an unprotected internal service can lead to data leaks, unauthorized access, or full system compromise.
