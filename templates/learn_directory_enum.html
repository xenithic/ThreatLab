Directory Enumeration Vulnerability

Concept

Web servers may contain hidden directories not linked on the website.

Although they aren't visible or accessible from the homepage, if the files physically exist on the server (like an admin panel), anyone knowing the path could reach them.

Tools Used for Directory Enumeration

Attackers use automated tools to discover hidden directories.

Example tools:

  • Gobuster
  • Dirb
  • FFUF

Example command:

gobuster dir -u http://target-site -w wordlist.txt

The tool tries many possible directory names using a wordlist.

Example discovered paths:

/admin-panel
/backup
/uploads

Attackers then visit the discovered path.

Example Attack

If a target website is located at:

http://example.com

Attackers may discover hidden paths like:

/admin
/backup
/secret

Real World Impact

Finding hidden directories might expose sensitive functionality or data, such as:

  • admin dashboards
  • backup files (.bak, .zip)
  • configuration files (.env, .git)
