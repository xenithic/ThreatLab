Profile Lookup Tool

This tool allows you to look up user profiles on the system.

Usage

Access profiles using the URL parameter:

/profile?user=<username>

Try it out — some example users might be: guest, user, john

Lab Instructions

Goal: Access a privileged profile and retrieve the flag.

Steps:

  1. Use the profile lookup tool via URL parameters
  2. Try different usernames
  3. Think about which user might have elevated privileges
  4. Retrieve the flag
  5. Submit the flag
